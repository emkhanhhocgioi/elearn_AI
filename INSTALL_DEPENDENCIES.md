# Cài đặt Dependencies cho Notification System

## Chạy lệnh sau trong terminal tại thư mục AI_FE/elearn_app:

```bash
npm install date-fns lucide-react
```

hoặc nếu dùng yarn:

```bash
yarn add date-fns lucide-react
```

## Packages được cài đặt:

1. **date-fns** - Thư viện format và xử lý thời gian
2. **lucide-react** - Icon library (có thể đã được cài rồi)

## Sau khi cài xong, restart dev server:

```bash
npm run dev
```
